# Termux Banner


Installation Step :

1) <code>git clone https://github.com/noobanon/Termux</code>


2) cd Termux-Banner


3) chmod +x requirement.sh


4) chmod +x t-ban.sh


5) bash requirement.sh


6) bash t-ban.sh



Uninstallation Step : (Remove Logo From Termux)

1) bash remove.sh

2) Done !!!
